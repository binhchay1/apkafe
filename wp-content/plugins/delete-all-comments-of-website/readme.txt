Delete all comments from your wordpress database easily.

== Description ==

The  'Delete All Comments of WordPress' plugin is built to delete all comments (Approved,Pending) from wordpress database by only one click and vary easily.

Use this Plugin under admin :

Tools > Delete Comments

== Installation ==

Just a standard installation required.

1. Plugins > Add New > Upload > 'delete-all-comments-of-wordpress-website.zip' - Install Now

2. Unzip & Upload 'delete-all-comments-of-wordpress-website' folder to the '/wp-content/plugins/' directory

3. Activate the plugin through the 'Plugins' menu in WordPress

4. Delete All Comments plugin is under 'Tools' menu
