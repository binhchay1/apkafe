== Frequently Asked Questions ==

 = Does the plugin come for free? =
 Yes, it's completely free.

 = Does the Plugin slow down your site? =
 In reality, The plugin does not make any difference to your visitors' experience of loading a page.

 = Is TOP Table Of Contents responsive? =
 Yep, it's fully responsive, and also looks & works great on smartphones.

 = Does it work with any WordPress theme? =
 Of course, it is compatible with every WordPress theme.

 = How to add TOP Table Of Contents to all articles automatically? =
 By default, It automatically inserts the table of contents to all posts. You can also use the shortcode on a specific post by turning off the auto insert options.

 = What is the shortcode for inserting TOC on any page? =
 Using [boomdevs_toc] shortcode, you can insert the TOC on any page.
